About this Server

Costi Survival World 9 is a Java based Minecraft survival server in collaboration with World of Cat server. It's main feature is that there is a role-play government where you are able to become your city's mayor if you win an election. Players will work together to build out their cities over time. The server will be run on little to no mods or plugins.
