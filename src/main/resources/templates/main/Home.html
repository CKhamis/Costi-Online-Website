About Me

New In Costi Online 9.0

The "Great Refactoring" update on my website has been a substantial project that started development in early July and has continued until now. Throughout this period, I've made several commits to enhance various aspects of the website. These updates include improving user creation, fixing issues with owner account creation, enhancing the functionality for viewing user accounts, and adding the ability to delete user accounts. I've also worked on profile picture options, made improvements in rendering notifications, and addressed UI issues.

This update has required extensive work on both the front-end and back-end of the website, with a primary focus on improving user account management. I've also made updates to features like Newsroom, LED Controller, and COMT functionality.
