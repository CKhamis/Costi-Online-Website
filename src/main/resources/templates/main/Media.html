About this Website

Costi Online is a website dedicated to centralizing my projects, services, history, and more. This website is hand coded my me using Java Spring and Java Hibernate. This website is open source, please see this website's associated GitHub page: Costi-Online-Website. Content will be continually added to this website over time. Please stay tuned for more.

The projects tab will contain links to see the various computer-related projects I have made over the years. These projects mostly cover websites and java based applications. Many of these projects are also available on my GitHub page.

The services tab will contain access to the various services I have. These services cover some of my skills such as device repair and server hosting. The Wiki tab has information on friend group inside jokes and other information about me personally. The media tab will have video based projects I have made over time. As listed there, many of these video based projects are uploaded to YouTube as well.

Finally, the Minecraft tab will take you to information surrounding my new Minecraft server. It will allow players of this world to participate in polls, elections, and learn more about its rich history.
